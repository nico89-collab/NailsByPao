# Nails By Pao

Sitio web comercial de manicuria con enfoque en conversion.

## Estructura

- `index.html`: pagina principal con hero, servicios, testimonios y footer.
- `soft-gel.html`
- `kapping.html`
- `semipermanente.html`
- `styles.css`: estilos globales, responsive, animaciones y componentes UI.
- `script.js`: menu mobile y animaciones al hacer scroll.

## Caracteristicas implementadas

- Diseno moderno, femenino y minimalista.
- Paleta en degrade rosado oscuro a blanco.
- Navegacion responsive con menu hamburguesa.
- Boton flotante de WhatsApp.
- Asistente virtual flotante con respuestas automaticas y acceso a WhatsApp.
- Animaciones suaves (hover + reveal on scroll).
- SEO basico (meta description, robots, Open Graph en home).
- Estructura estatica ideal para GitHub Pages o Vercel.

## Publicar en GitHub Pages

1. Subir el contenido de esta carpeta al repositorio.
2. En GitHub: Settings > Pages.
3. Seleccionar branch principal y carpeta root.
4. Guardar y esperar la URL de despliegue.

## Publicar en Vercel

1. Importar el repositorio en Vercel.
2. Framework preset: Other.
3. Build command: vacio.
4. Output directory: `.`
5. Deploy.

## Personalizacion rapida

- Numero de WhatsApp: buscar `2657603395` en todos los HTML y reemplazar.
- Redes sociales: reemplazar links de Instagram/Facebook/TikTok en footer.
- Precios y textos: editar cada subpagina segun tu lista real.
